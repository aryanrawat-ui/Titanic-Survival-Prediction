# Titanic Survival Prediction System 🚢

## About the Project

This project is a simple machine learning classification project built using the famous Titanic dataset. The main goal is to predict whether a passenger would survive or not based on details like age, gender, passenger class, fare, and embarked location.

I created this project to strengthen my understanding of machine learning basics like data cleaning, preprocessing, model training, and prediction. It also helped me understand how real-world datasets are handled before applying ML models.

---

## What I Did

* Loaded and explored the Titanic dataset
* Handled missing values in Age and Embarked columns
* Removed unnecessary columns like Cabin, Name, Ticket, and PassengerId
* Converted categorical columns like Sex and Embarked into numerical values
* Performed basic Exploratory Data Analysis (EDA)
* Trained a Logistic Regression model for prediction
* Evaluated model performance using Accuracy Score and Confusion Matrix
* Saved the trained model using Joblib
